# 汽车租赁管理系统 java325

## 项目概述

本项目是基于 Springboot 与 Vue 的前后端分离的汽车租赁管理系统。系统提供了一套完整的论文样例，适用于学习和研究之用。

## 技术栈

- 后端：Spring Boot
- 前端：Vue.js

## 功能模块

- 用户管理
- 车辆管理
- 租赁订单处理
- 费用结算
- 数据统计分析

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+
- 前端：Node.js 12.0+

## 部署步骤

1. 准备后端所需环境：Java 1.8+，MySQL 5.7+
2. 导入数据库文件
3. 编译并运行后端服务
4. 准备前端所需环境：Node.js 12.0+
5. 前端项目编译打包
6. 部署至 Web 服务器
